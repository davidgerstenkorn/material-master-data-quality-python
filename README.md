# Material Master Data Quality (Python)

This project implements a Python-based data quality pipeline for validating a synthetic ERP-like material master dataset.

The pipeline detects common data quality issues such as:

- missing material groups
- missing units of measure
- negative stock values
- missing lead times
- duplicate material IDs
- invalid reorder points

## Tech Stack

- Python
- pandas
